# EnergyMonitor with Arduino

Medidor de consumo electico con reporte en tiempo real

## Tecnologias utilizadas: 

  + Arduino
  + AngularJS
  + NodeJS
  + ExpressJS
  + MongoDB
  
## Equipo utilizado:
  + Arduino uno
  + Energy Monitor Shield v0.1 para Arduino
  + Sensor de corriente SCT-013
  + Ethernet Shield para Arduino
